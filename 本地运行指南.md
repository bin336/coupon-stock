# 券库 · 本地电脑运行指南

把你录入的券和图片都带上了（在 `data/` 文件夹里），解压后按下面任意一种方式跑起来即可。

## 方式一：Docker 运行（最省心，强烈推荐）

适合：不想装 Node、怕原生编译报错的人。
前置：电脑装好 Docker Desktop（Windows / Mac 官网一键装，约 5 分钟）。

1. 解压本包，进入 `coupon-stock` 文件夹
2. 在文件夹里打开终端/命令行，执行：
   ```
   docker compose up -d
   ```
3. 浏览器打开 `http://localhost:3000`
4. 默认管理员账号：`admin` / `admin123`
   ⚠️ 上线前务必改密码：复制 `.env.example` 为 `.env`，改 `ADMIN_PASS` 和 `JWT_SECRET` 为你自己的，再重跑上面的命令。

## 方式二：Node.js 直接运行

适合：电脑已装 Node.js 18+。
1. 进入 `coupon-stock` 文件夹
2. 命令行执行：
   ```
   npm install
   npm start
   ```
3. 打开 `http://localhost:3000`

> 如果 `npm install` 时 `better-sqlite3` 报错（缺少编译环境），别折腾，直接用「方式一 Docker」即可，完全避开原生编译。

## 数据在哪 / 怎么备份

- `coupon-stock/data/coupon.db` ：库存数据库
- `coupon-stock/data/uploads/` ：所有二维码截图

**备份** = 复制整个 `data` 文件夹。
**换电脑 / 迁 NAS** = 把整个 `data` 文件夹拷过去覆盖即可，券和图片全在。

## 以后要上 NAS（24 小时运行）怎么办

代码已经是 Docker 化的，NAS 上只要有 Docker（群晖「Container Station」/ 联昊通对应容器功能），步骤和上面「方式一」完全一样：
拷贝 `coupon-stock` 文件夹到 NAS → `docker compose up -d` → 浏览器访问 NAS 的 IP:3000。
这一步等你知道 NAS 怎么用、且功能都定稿后再做，最省事。
