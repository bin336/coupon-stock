# 券库（优惠券库存管理）使用文档

> 适用版本：v3.35（已部署 NAS，截至 2026-07-19）
> 代码当前领先：含 **v3.36 待部署**改动（gzip 压缩 + 缓存 + docker-compose 默认 host 模式），尚未上 NAS。见《待部署记录.md》。
> 用途：囤优惠券、按券管理、卖券结算与对账。手机/电脑浏览器均可使用，适配微信群运营场景。

---

## 一、登录与账号

- 默认管理员账号：`admin / admin123`，**首次部署后务必修改**。
- 角色分为 **管理员** 与 **普通用户**：
  - 普通用户：可录入券、查看/结算自己的券；**看不到他人账务**。
  - 管理员：在普通用户权限基础上，额外可使用报表、查看全部结算、管理用户。
  - **普通用户现在也能进入「设置 → 个人资料」**：可修改**自己的**昵称与密码（仅限本人，看不到也无法改动他人）。

---

## 二、录入券

入口：首页右下角 **「+」浮动按钮** → 选择「快速入库」（单条）或「批量入库」。

### 单条录入
| 字段 | 说明 |
|------|------|
| 商家名称 * | 必填。输入时下拉自动补全历史商家名，避免「星巴克」/「星 巴克」重名分裂 |
| 代金券金额 | 单张面值 |
| 张数 | 同一条目包含几张 |
| 券号 | 选填 |
| 过期时间 * | 必填 |
| 成本 | **选填**（v3.30 起改为选填，与批量一致） |
| 所有人 | 默认当前登录人 |
| 平台 | 如：点评 / 抖音（选填） |
| 备注 | 选填 |
| 二维码截图 | 选填。上传后自动 OCR 识别商家/券号/过期时间，识别到的日期会自动填入 |

- **同商家再录**：录入弹窗内点「同商家再录」，保留商家/过期/所有人/平台，只清空金额与张数，方便快速录多面值（如美团 100 → 50 → 20）。

### 批量入库
一次选多张二维码截图，先填「通用信息」点「应用到全部」，再逐张微调；提交后按商家+面值汇总入库。

---

## 三、搜索

首页顶部搜索框，支持按 **商家 / 券号 / 所有人 / 平台 / 备注 / 面值** 检索。

- **分词搜索**：空格分隔多个条件为「与」关系，例如 `许家菜 100` 表示「许家菜商家的 100 元面值券」。
- **中文+数字自动补空格**：`许家菜100` 与 `许家菜 100` 等价（手机连打不带空格也能搜）。
- 搜索结果底部显示数量提示，例如「搜索「许家菜」找到 7 张券」。

---

## 四、汇总视图

首页「📊 按商家」/「👤 按所有人」按钮进入汇总：

- 同一分组下**各面值以标签呈现**（如 `¥100 ×3 / ¥50 ×2`），一眼看清不同面值各多少张。
- 点击某个**面值标签**可下钻到「该分组 + 该面值」的明细列表。
- 汇总页内**搜索框**可按分组键（商家名/所有人名）实时过滤卡片。

---

## 五、结算与报表

- **待结算**：首页「待结算」卡片点击进入（**所有人均可进入，只看自己的**）；管理员可见全部。
- 流程：标记售出 → 标记结算。结算金额 = 售出价 − 售出价 × 1.6%。
- **报表**（仅管理员）：利润 / 售出统计，支持按所有人、时间段筛选。

---

## 六、操作日志

「设置 → 操作日志」查看谁在何时做了什么（录入 / 售出 / 结算 / 删除 / 用户管理等），便于对账。

- 单条录入日志：对象=商家名，详情=`面值¥100 ×2张 成本¥… 所有人:…`。
- 批量入库日志：对象=真实商家名（多商家取前 3 个并标注家数），详情=逐券「商家 ¥面值 ×张数」摘要。

---

## 七、图片：存图与分享

> ⚠️ 网页安全限制：**网页不能自动把图片写入手机相册**，也不能直接打开某个闲鱼聊天框。以下是当前能做的上限。

- **缩略图「分享」按钮**（右下角）：调起系统分享面板，选「闲鱼」后选会话发送（带图+券文案）。微信内置浏览器内系统分享可能受限，此时会**自动复制券文案并打开图片**，你手动去闲鱼粘贴即可。
- **「复制信息」按钮**（卡片底部）：一键复制券文案（商家/面值/券号/有效期），去闲鱼粘贴发送。
- **大图视图「保存到手机」**：打开大图后**长按图片**即可保存到相册（iOS / 安卓通用）。

---

## 八、用户管理（管理员）

「设置 → 用户管理」可新增/删除/编辑用户、修改密码。

- 保护规则：不能删除当前登录账号；不能删除最后一个管理员。
- 删除用户前会显示其名下券数量，可转移给接手人或保留为无主。

---

## 九、已知限制（部署/使用须知）

1. **存图需长按**：网页无法直接存相册，必须长按保存（或微信内用 JS-SDK 可实现一键存，需另行接入）。
2. **分享到闲鱼需手动选**：网页无法跳过「选闲鱼 + 选会话」两步；微信内分享只能到微信，到不了闲鱼。
3. **外网访问建议加 HTTPS**：账号密码在 HTTP 下不安全，外网暴露请配合反向代理 + 证书。
4. **数据备份**：整个 `coupon-stock` 目录即数据库（含 `./data`），备份=复制该目录。

## 十、手机安装 Tailscale 上公网（详细教程）

> 目的：不在家里 WiFi（用流量）也能开券库。Tailscale 免费、不用碰路由器、全程加密、免备案。
> 前提：NAS 上已装好 Tailscale 并登录（见《NAS上公网步骤》）。本节约教你**手机端怎么装、商店搜不到怎么办**。

### 1）苹果 iPhone / iPad

1. 打开 **App Store**，搜索 **Tailscale**（图标是蓝色小尾巴）。
2. 搜不到？多半是 Apple ID 地区问题，二选一：
   - 直接用这个链接打开：👉 `https://apps.apple.com/app/tailscale/id1470499027`
   - 或把 Apple ID 地区切到「美国 / 香港」等再搜（设置 → 头像 → 媒体与购买项目 → 查看账户 → 国家/地区）。
3. 下载安装 → 打开 App（登录与连接见下方「3）登录账号」）。

### 2）安卓 Android（国内常搜不到，重点看这里）

**方式 A：Google Play（有谷歌服务的手机）**
- 打开 **Google Play**，搜 **Tailscale** → 安装。

**方式 B：官方 APK（搜不到 / 没谷歌商店时，推荐）**
国内很多安卓手机没有 Google Play，直接用 Tailscale 官方签名安装包：
- **GitHub 官方发布页**（最稳）：👉 `https://github.com/tailscale/tailscale-android/releases`
  拉到最新版本，下载 `app-universal-release.apk`（带 `universal` 字样的那个，通吃各品牌）。
- **F-Droid 商店**（另一个官方渠道）：👉 `https://f-droid.org/zh_Hans/packages/com.tailscale.ipn/`
  点「下载 APK」或先装 F-Droid 再搜 Tailscale。
- 下载完点开 APK → 按提示**允许「未知来源」安装** → 安装。（这两个都是 Tailscale 官方签名，安全，不是来路不明的包。）

### 3）登录账号（重要，关系到能不能登得上）

打开 App 后要点「登录 / Log in」：
- ✅ **推荐用 Microsoft（Outlook / Hotmail）账号**，或选 **「用邮箱登录 / Email」+ 收魔法链接**。
- ⚠️ **尽量别用 Google 账号**：国内网络常被墙，登录会卡住转圈。
- **NAS 和手机必须用同一个 Tailscale 账号**（两边登录同一个），否则互相看不到。

### 4）连接 + 打开券库

1. 手机上把 Tailscale 的开关打开（显示「已连接 / Connected」）。
2. 在 Tailscale App 里能看到你的 NAS 设备，点进去能看到它的 **Tailscale IP**（形如 `100.x.x.x`，记下来）。
3. 手机浏览器（任意浏览器）打开：
   ```
   http://100.x.x.x:3000
   ```
   （把 `100.x.x.x` 换成你 NAS 的 Tailscale IP）
4. 看到登录页 = 成功！用 `admin` + 你设的密码登录。

> 以后手机只要开着 Tailscale，走到哪都能开券库，和你在家一样。NAS 的局域网 IP 天天变也不影响它。

### 5）商店搜不到的兜底方案汇总

| 设备 | 正常下载 | 搜不到 / 用不了时的办法 |
|------|----------|------------------------|
| iPhone / iPad | App Store 搜 **Tailscale** | 用直链 `https://apps.apple.com/app/tailscale/id1470499027`；或把 Apple ID 地区切到美/港再搜 |
| 安卓 Android | Google Play 搜 **Tailscale** | 用官方 APK：GitHub Releases（`tailscale-android`，下 `app-universal-release.apk`）或 F-Droid（`com.tailscale.ipn`）；装前允许「未知来源」 |

> 两个官方 APK 渠道都是 Tailscale 官方签名，安全。装好后在 App 里登录（见第 3 步），开着开关就能用。

## 十一、版本更新要点（v3.10 → v3.35 关键变更）

- 普通用户可进入待结算模块。
- 按商家 / 按所有人汇总视图 + 面值下钻。
- 商家名自动补全（自定义下拉，修复移动端遮挡）。
- 防重复提交（提交瞬间锁定按钮，连点只生效一次）。
- 成本改为选填；批量入库操作日志修正。
- 搜索中文+数字自动补空格。
- 缩略图「分享」上移、「存图」取消；移除存文件逻辑。
- **v3.34**：分享兜底取消「已复制…粘贴发送」提示文字；清理 `saveImage` 废弃参数；新增本使用文档。
- **v3.35**：大图视图移除「保存到手机」、按钮「分享到闲鱼」改名为「分享」；普通用户可进入「设置 → 个人资料」仅修改自己的昵称与密码（操作日志对其隐藏，用户管理对其仅开放本人资料）。
- **v3.36（待部署，尚未上 NAS）**：
  - 后端启用以 gzip 压缩（app.js 体积压缩约 72%），静态资源缓存 10 分钟、二维码截图缓存 7 天（HTML 不缓存，更新即时生效）；docker-compose 默认改用 `network_mode: host`，Tailscale 外网访问开箱即用（提交 `80f246e` + `ed43944`）。
  - **重复录入拦截**：券入库前按「券号 + 图片 MD5 指纹」与未售券比对，疑似重复时弹窗确认，确认后可强制录入（单条/批量均支持），防止 100+ 图里录重导致对账错。
  - **今日运营卡片**：首页新增「今日运营」，一眼看今日录入/售出/结算额 + 库存总值 + 7 天内到期，开局即有全局视图（提交 `bef9dfc`）。
  - **OCR 提速**：二维码识别改用「复用单例 worker + 启动预热」，单张识别从 5–10 秒降至约 1 秒（约 5–10 倍），录入效率显著提升（提交 `80944cc`）。
  - **多图批量录入·紧凑卡片**：批量弹窗每张图改为紧凑卡片——默认收起 8 个输入框，仅显示缩略图 + 一行识别摘要（商家·面值·券号·过期·张数）+ 识别状态（🤖 识别中… / ✓ 识别 N 项 / 未识别请手填），点「展开」才按需编辑；字段改动实时刷新摘要，缺商家名时变红提醒。一次选 10+ 张不再堆出上百输入框、滚动找券（提交 `27f3aa9`）。
  - **通用填写新增平台/备注**：批量弹窗「通用填写」增加「平台」「备注」输入，与「成本」同排一行；「应用到全部」一并套用平台与备注（提交 `867c0cc`）。
  - **Bug 修复**：今日运营卡片金额修复双 ¥ 符号显示（`8a813f5`）；`loadData` 三接口独立容错，单接口异常不再导致整页数据全挂（提交 `8a813f5`）。
